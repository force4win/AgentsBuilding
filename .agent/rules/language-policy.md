# Language Policy

Este proyecto siempre debe utilizar el español como idioma principal para todas las interacciones.

## Reglas de Idioma
1. **Respuestas al Usuario**: Todas las respuestas directas deben ser en español.
2. **Razonamiento (Thoughts)**: El bloque de razonamiento interno del asistente debe estar en español.
3. **Resúmenes de Tareas**: Los nombres de las tareas, estados y resúmenes en `task_boundary` deben ser en español.
4. **Documentación**: La documentación generada por el asistente (como `walkthrough.md`, `implementation_plan.md`, `task.md`) debe estar en español, a menos que el código fuente requiera lo contrario (nombres de variables, etc.).
