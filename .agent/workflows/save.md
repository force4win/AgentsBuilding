# Save Project State
Activation: Manual (/save)

## Steps
1. **Status Discovery**: into directory `.alvhere/` Read the current `task.md` and review the conversation history to summarize the latest accomplishments and pending items.
2. **Project Documentation**: Update or create the following files into `.alvhere/` directory:
   - `README.md`: Ensure it contains the current mission, tech stack, and key features.
   - `HANDOFF.md`: Detailed transition notes for the next agent/session, including critical context and next tasks.
   - `PROJECT_CONTEXT.md`: Detalle de todo lo que la IA necesite recordar para continuar mas adeleante, inclusive podria guardar en su idioma
   - `GUIDELINE.md`: mision contruida del proyecto
3. **Session Summary**: Generate a concise summary of architectural decisions and lessons learned.
4. **Final Sync**: Ensure all artifacts and progress trackers are up to date.
5. **Report**: Confirm that the state has been persisted.
