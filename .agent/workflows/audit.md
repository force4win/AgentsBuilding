# Audit Code (Universal)
Description: Performs a high-level diagnostic of security, performance, and readability.
Activation: Manual (/audit)

## Steps
1. **Analyze File**: Read the active file and understand its core logic.
2. **Security Audit**: Check for common vulnerabilities (e.g., SQL injection, insecure storage).
3. **Performance Check**: Identify potential bottlenecks or inefficient loops.
4. **Readability Check**: Verify against Clean Code and Documentation rules.
5. **Output**: List 3 key findings and 1 "Quick Win" refactoring.
