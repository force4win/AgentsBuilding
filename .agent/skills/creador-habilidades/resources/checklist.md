# Lista de Verificación de Nueva Habilidad

Antes de finalizar la creación de una habilidad, asegúrate de cumplir con:

- [ ] La carpeta existe en `.agent/skills/<nombre-habilidad>`.
- [ ] El archivo `SKILL.md` tiene el frontmatter YAML correcto (`name`, `description`).
- [ ] El contenido de `SKILL.md` está íntegramente en español.
- [ ] No hay errores de sintaxis en el Markdown.
- [ ] Se han incluido los recursos o scripts necesarios.
- [ ] Se ha notificado al usuario sobre la nueva capacidad disponible.
